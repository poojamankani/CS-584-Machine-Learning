Steps to execute the program:
1. Make sure that you have IPython notebook, ELM pacakage and CVXOPT package installed on your machine.
2. Make sure that the actual_data.csv, actual_data_space.csv and test_input.txt is present in the same directory. You can modify the 
   test_input.txt to check on different input data.
3. Open the project.ipynb file in ipython notebook, and click "run all".
4. Intermediate results and performance graphs will be generated in the ipynb output cells.
5. For the predicted output on the input data provided, please check the last output cell.
Note: The program is computation intensive, and takes a while to execute completely.
